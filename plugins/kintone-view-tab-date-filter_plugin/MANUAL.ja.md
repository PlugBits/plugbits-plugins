# PlugBits ListKit 操作マニュアル

## 1. できること

このプラグインは、kintone の一覧画面に次の機能を追加します。

- ビュータブ
- 日付フィルター
- 集計表示

## 2. 設定画面の開き方

【スクリーンショット挿入位置: アプリ設定からプラグイン設定を開く画面】

1. 対象アプリを開きます。
2. アプリ設定を開きます。
3. プラグイン設定で `PlugBits ListKit` を開きます。

## 3. ビュータブ

【スクリーンショット挿入位置: ビュータブ設定画面全体】

`ビュータブを有効化` をオンにすると、一覧上部にビュー切替タブを表示します。

- `利用可能なビュー`
  - タブに出せるビューの一覧です。
- `表示順`
  - 実際に表示するビューの一覧です。
- `追加 ->`
  - 選んだビューを表示対象に追加します。
- `<- 削除`
  - 表示対象から外します。
- `上へ / 下へ`
  - タブの順番を変更します。
- `横並びタブ数`
  - 画面に直接出すタブ数です。超えた分は `その他` に入ります。
- `アイコン表示`
  - ビュー種別のアイコンを表示します。

## 4. 日付フィルター

【スクリーンショット挿入位置: 日付フィルター設定画面全体】

`日付フィルターを有効化` をオンにすると、一覧上部に日付フィルターを表示します。

- `対象日付フィールド`
  - 絞り込みに使う日付または日時フィールドです。
- `週の開始`
  - `今週` `先週` の基準です。
- `表示プリセット`
  - 利用者に表示する期間ボタンを選びます。
- `既定プリセット`
  - 最初に選ばれる期間です。
- `表示対象ビュー`
  - 日付フィルターを表示するビューを選びます。未選択なら全ビューです。

## 5. 集計

【スクリーンショット挿入位置: 集計設定画面全体】

`集計を有効化` をオンにすると、一覧上部に集計結果を表示します。

- `表示対象ビュー`
  - 集計を表示するビューを選びます。未選択なら全ビューです。
- `行を追加`
  - 集計ルールを追加します。

各行で設定する内容:

- `フィールド`
  - 集計する数値フィールドまたは計算フィールド
- `集計`
  - 合計 / 平均 / 最小 / 最大 / 件数
- `小数桁`
  - 小数点以下の表示桁数
- `接頭`
  - 数字の前に付ける文字
- `接尾`
  - 数字の後に付ける文字

## 6. 一覧画面での使い方

【スクリーンショット挿入位置: 一覧画面上部にタブ・日付フィルター・集計が表示されている画面】

### ビュータブ

【スクリーンショット挿入位置: ビュータブ表示例】

- タブをクリックするとビューを切り替えます。
- タブが多い場合は `その他` から選びます。

### 日付フィルター

【スクリーンショット挿入位置: 日付フィルターのポップアップ表示例】

1. `日付` をクリックします。
2. 期間を選ぶか、開始日と終了日を入力します。
3. `適用` を押します。
4. 条件を外すときは `クリア` を押します。

### 集計

【スクリーンショット挿入位置: 集計表示例】

- 一覧上部に設定した集計結果が表示されます。
- 表示内容は、現在のビューと絞り込み条件に応じて変わります。

## 7. よくある確認ポイント

### 日付フィルターが出ない

- `日付フィルターを有効化` がオンか
- 対象ビューに含まれているか
- 日付または日時フィールドがあるか

### 集計が出ない

- `集計を有効化` がオンか
- 集計ルールが 1 行以上あるか
- 数値または計算フィールドを選んでいるか

### 動作が重い

- 集計対象のビューを減らす
- 集計行を減らす
- 必要に応じて集計をオフにする

## 8. 注意

- 設定変更後は、必要に応じてアプリを更新してください。
- 一覧件数が多い場合、集計表示に時間がかかることがあります。
