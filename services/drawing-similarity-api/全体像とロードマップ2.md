# kintone Drawing Similarity：現状と今後のロードマップ

最終更新: 2026-07-01

---

## 目的

kintone に登録された図面 PDF を対象に、類似図面検索と図面登録支援を実装する。

類似検索は画像 Embedding だけでなく以下を組み合わせる。

- 画像 Embedding 検索（DINOv2 / OpenCLIP）
- OCR 文字抽出
- 図番・品名・材質・寸法などの項目抽出
- kintone 項目・抽出項目による絞り込み／スコア補正
- 形状特徴量（shape features）によるスコア補正

---

## システム構成

```
kintone Plugin
  ↓  REST API (X-API-Key)
Cloud Run API（GCP: drawing-similarity-501101）
  ↓                  ↓
Qdrant Cloud     Firestore（テナント認証）
                 Vertex AI（OCR / Gemini）
```

### Cloud Run API（services/drawing-similarity-api）

| エンドポイント | 機能 |
|---|---|
| GET /health | 死活確認 |
| POST /index | PDF 取得 → PNG 化 → Embedding → OCR → Qdrant upsert |
| POST /similar | Qdrant 類似検索 → スコアキャリブレーション → 返却（recordId / fileKey / pdf_base64 いずれのクエリにも対応） |
| POST /analyze | PDF → OCR → 項目抽出（kintone に保存しない） |
| POST /tag | タグ同期（Qdrant payload 更新） |
| GET /tags | テナント内タグ一覧取得 |
| GET /field-values | テナント内フィールド値サジェスト取得 |

| 機能 | 備考 |
|---|---|
| DINOv2 埋め込み（現在使用中） | EMBEDDING_PROVIDER=dinov2 |
| OpenCLIP 埋め込み | EMBEDDING_PROVIDER=openclip |
| Tesseract OCR | 日本語＋英語 |
| 形状解析（shape features） | bbox・輪郭・ink ratio など |

### 機能追加：手元の図面でその場検索（一覧画面）【実装済み】

**概要**

kintone に登録せずに、手元にある PDF をアップロードしてその場で類似図面を検索できる機能。

**ユースケース**

- 見積依頼を受けた図面が既存図面と似ているか素早く確認したい
- kintone に登録する前に類似品を調べたい

**想定フロー**

```
一覧画面「類似検索（アップロード）」ボタン
  ↓
PDF drag & drop モーダル
  ↓
/similar に pdf_base64 を直接送信（recordId 不要）
  ↓
類似結果をモーダルに表示（既存の類似検索 UI を流用）
```

**実装内容**

- plugin.js：一覧画面に「類似検索（アップロード）」ボタンを追加。`openUploadSimilarModal` で drag & drop → base64 化 → `/similar` 呼び出し → 既存の `renderSimilarList` で結果表示。
- server.js：`/similar` に `pdf_base64` パスを追加（recordId なしでその場で PNG 化 → shape profile + Embedding を生成して検索、Qdrant には保存しない。`queryVectorSource = 'uploaded-pdf'`）。
